# Installation

In your 'Preferences ━ Browse Packages...' folder

```bash
git clone https://github.com/moodmosaic/hasky.git Hasky
```

## Monochromic file icons (optional)

```bash
git clone -b mono https://github.com/braver/FileIcons.git
```
